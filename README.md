## Description

Vinyl Vault is a full-stack MERN application that allows music enthusiasts to catalog, manage, and share their vinyl record collections. Users can create an account, add albums to their personal vault with automatic artwork search via the iTunes API, write and read reviews with ratings (1-10), and explore other users' collections in the community page.

- üéµ **Personal Collection Management:** Full CRUD operations on your album collection

- üë§ **User-Based Organization:** Browse collections organized by user
- üé® **Vintage Aesthetic:** Beautiful brown/tan theme inspired by vinyl records

### Why I Built This

As a music enthusiast, I wanted to create a digital space where collectors can not only catalog their own vinyl collections but also share their passion with others. This app combines personal collection management with social features, allowing users to discover new music through community reviews and recommendations.

- **Frontend Framework:** React 19.2.0
- **Routing:** React Router DOM 7.x

- **Styling:** CSS3 with CSS Grid, Flexbox, and backdrop filters
- **Authentication:** JWT (JSON Web Tokens)
- **State Management:** React Context API
- **External API:** iTunes Search API for album artwork
